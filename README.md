# E-commerce JS Project

A simple e-commerce practice project built with HTML, CSS, and JavaScript.
It uses a local API with json-server and a db.json file.

## Features

- Display products from API
- Create new products
- Delete products
- Local data storage with db.json

## Technologies

- HTML
- CSS
- JavaScript
- json-server

## Project Structure

project/

│   └── create.js

    └── style.css

## How to Run

1. Open the project in VS Code
2. Start the local API:
   npm run server
3. Open index.html with Live Server

API runs on:
http://localhost:3000

## Notes

- This project uses a local server (json-server), so it must be run locally
- Data is stored in db.json
- Make sure the server is running before using the app
